# Claude Agentic AI SDLC Curriculum Chat

## Summary

This Claude shared chat captures a curriculum-design exchange for [[wiki/entities/Cheersin Interactive|Cheersin Interactive]], focused on advanced Agentic AI across the SDLC. Claude proposed a 12-session, 24-hour cohort covering foundations, core stack, SDLC agents, and governance; the user then pushed back that the program should be structured in 1-hour blocks.

## Key Claims

- The client wants implementation-driven training, not generic AI awareness.
- The participant mix spans developers, BAs, PMs, QA, data, UX, and senior leaders, so the program needs shared foundations plus role-aware application.
- The tool stack includes [[wiki/entities/LiteLLM|LiteLLM]], [[wiki/entities/Ollama|Ollama]], [[wiki/entities/Weaviate|Weaviate]], [[wiki/entities/Langfuse|Langfuse]], [[wiki/entities/Factory AI|Factory AI]], and [[wiki/entities/CrewAI|CrewAI]].
- The 70% operating-cost reduction target should be treated as aspirational, not promised by training alone.
- The user's later correction suggests the curriculum should be modular enough to work as 1-hour learning blocks.

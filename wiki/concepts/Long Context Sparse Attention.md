# Long Context Sparse Attention

## Definition

Long context sparse attention refers to attention architectures that reduce the cost of attending over very large context windows by making attention patterns sparse rather than fully quadratic.

## Why It Matters

Sparse attention could make very long context windows more practical, but claims about speed, quality, and frontier-model capability need careful verification.

## Example

- [[wiki/entities/SubQ|SubQ]] claims a fully sub-quadratic sparse-attention architecture with a 12 million token context window.

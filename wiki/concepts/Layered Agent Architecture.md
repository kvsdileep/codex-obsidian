# Layered Agent Architecture

## Definition

Layered agent architecture separates an agent system into distinct control surfaces such as memory, modular expertise, deterministic guardrails, delegation, and distribution.

## Why It Matters

Prompting alone is too soft for reliable production work. Layering makes responsibilities explicit and lets teams improve the runtime around the model.

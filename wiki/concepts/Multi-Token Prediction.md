# Multi-Token Prediction

## Definition

Multi-token prediction is a model generation technique where the system predicts multiple future tokens in one step rather than producing only the next token.

## Why It Matters

If quality is preserved, multi-token prediction can make LLM responses noticeably faster without requiring users to switch models or accept weaker output.

## Example

- [[wiki/entities/Gemma|Gemma]] announced MTP drafters that it says can triple output speed.

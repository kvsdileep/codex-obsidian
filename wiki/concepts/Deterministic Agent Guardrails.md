# Deterministic Agent Guardrails

## Definition

Deterministic agent guardrails are runtime controls that intercept, block, rewrite, or audit agent actions outside the model's discretion.

## Why It Matters

Instructions can be ignored or misapplied. Deterministic controls are useful for protected files, formatting, risky commands, context trimming, and audit trails.

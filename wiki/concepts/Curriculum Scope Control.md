# Curriculum Scope Control

## Definition

Curriculum scope control is the practice of sizing learning blocks so a program remains actionable for the audience and delivery format.

## Why It Matters

The Claude chat shows a common risk: a curriculum can be technically thoughtful but too heavy for the buyer's preferred format. The user's correction toward 1-hour blocks is a scope signal.

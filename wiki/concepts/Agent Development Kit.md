# Agent Development Kit

## Definition

An agent development kit is a set of primitives for shaping, constraining, extending, delegating, and distributing agent behavior.

## Why It Matters

The clipped article argues that Claude Code already behaves like an agent development kit because it exposes memory files, skills, hooks, subagents, plugins, and MCP.

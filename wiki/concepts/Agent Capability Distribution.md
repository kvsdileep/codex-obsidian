# Agent Capability Distribution

## Definition

Agent capability distribution is the packaging of agent behavior, tools, skills, settings, and conventions so they can be reused across projects and teams.

## Why It Matters

Without distribution, agent improvements stay local to one repo or one person's setup. Plugins and shared configurations turn agent behavior into team infrastructure.

# Conventions

## Tags

Use a small tag set:

- `source`
- `concept`
- `entity`
- `topic`
- `question`
- `meta`

Add domain tags only when they help retrieval.

## Status

- `seed`: newly created and lightly processed
- `developing`: useful but incomplete
- `mature`: reliable and well linked
- `evergreen`: core vault infrastructure or stable knowledge
- `archived`: no longer active but kept for record

## Links

Use wikilinks for internal references. Prefer `[[Concept Name]]` over path links.

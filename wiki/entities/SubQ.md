# SubQ

## Overview

SubQ is described in a tweet by Alexander Whedon as a model built on a fully sub-quadratic sparse-attention architecture.

## Key Facts

- Claimed 12 million token context window.
- Claimed 52x faster than FlashAttention at 1 million tokens.
- Needs independent verification before treating the claims as settled.

# Codex Quick Start

Use this vault from Codex or Slack with simple commands.

## Ingest

```text
ingest everything new in .raw
ingest new web clips
ingest tweets
ingest the latest chat export
```

Codex will preserve raw files and write synthesized notes into `wiki/`.

## Ask

```text
what do we know about creator funnels?
find connections between AI agents and course operations
summarize recent clips about marketing
```

Codex should read `wiki/hot.md`, search `wiki/index.md`, then inspect relevant notes.

## Maintain

```text
run a vault health check
find orphan notes
update the hot cache
```

Keep the vault useful, small, and well linked.
